# Project Title

Warranty Claim – A Soroban Smart Contract for Product Warranty Tracking on Stellar

## Project Vision

This project demonstrates a **product warranty tracking system** built on Soroban and Stellar. It enables manufacturers to register products and warranty periods, owners to register their ownership and file warranty claims, and manufacturers to resolve those claims on-chain.

The goal is to provide a transparent, decentralized warranty management system where all parties can verify product authenticity, ownership, and claim status permanently on the blockchain.

---

## Description

A Soroban smart contract dApp that manages product warranties on Stellar. Manufacturers can register products with warranty periods, owners can register their ownership and file claims, and manufacturers can approve or reject claims — all with full on-chain transparency.

---

## Features

### 1. Product Registration
- Manufacturer registers product with unique ID and warranty duration
- Warranty tracking stored permanently on-chain

### 2. Ownership Registration
- Product owner registers their ownership linked to a product
- Ownership history transparent and verifiable

### 3. Warranty Claims
- Owner can file a claim with description
- Manufacturer can approve or reject claims
- Claim status tracked on-chain

### 4. On-chain Transparency
- All product, ownership, and claim data stored permanently
- Anyone can query product info and claim history

## Future Scopes

- Allow warranty transfer when product is sold

- Integrate with oracles for automated claim approval based on defined rules
